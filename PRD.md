# 秩云智能订单管理系统 V2.0 · 产品需求文档（PRD）

> **产品代号**：ZhiYun Order Intelligence V2.0
> **版本**：V2.0.6（演示原型）
> **文档来源**：基于 `index.html` 演示原型逆向提取
> **目标行业**：服装 / 制衣工厂（兼顾五金、家纺等行业模板）
> **核心口号**：让生产管理更简单、更智能 — 从一张订单到一件成衣，AI 让数据驱动每一个决策

---

## 1. 产品概述

### 1.1 一句话定位
面向中小型服装工厂的 **「订单驱动 · AI 赋能 · 千人千面 · 全流程协同」** 一体化智能订单与生产管理 SaaS。

### 1.2 关键指标（演示态）
| 指标 | 数值 |
| --- | --- |
| 业务模块数 | 14 |
| 页面响应 | < 3s |
| 系统可用率 | 99.9% |
| AI 预测准确率 | 86% |
| AI 效率提升 | +18% |
| 安全等级 | 等保三级 |
| 单工厂演示数据 | 12 订单 / 8 物料 / 4 采购 / 8 审批 / 6 预警 |

### 1.3 核心价值主张
1. **一张订单跑到底** — PO 号全程追溯（订单→采购→裁剪→缝制→后整→包装→出货）。
2. **AI 决策中枢** — 产能预测、智能排产、瓶颈预警、价格波动监控。
3. **千人千面** — 同一套系统按角色（厂长、计划员、业务员、仓管、裁剪负责人、超级管理员）裁剪菜单与数据视图。
4. **三秒直达** — 全局命令面板（⌘K）+ 12 组快捷键（G+字母 跳模块）。

## 4. 功能模块详述

### 4.1 ① 数据看板（Dashboard）
**定位**：生产运营全景驾驶舱，登录后默认页。

**功能要点**
- 顶部欢迎条：工厂代号 + 实时时钟（秒级刷新）+ 系统状态。
- **生产流程全景图**：8 节点可点击跳转（订单中心 → 采购计划 → 仓库入库 → 裁剪 → 缝制 → 后整 → 包装 → 出货）。
- **4 个核心 KPI**：本月订单总数、已延期订单、准时交付率、库存物料种类。
- **产线实时监控**（4 个工序组）：裁剪 A / 缝制 B / 后整 C / 包装 D，每组展示日产量、效率、趋势箭头；下方 7 日产能折线图（Chart.js）。
- **AI 排期日历**：未来关键节点卡片（出货截止、缝制完成、开裁、交期）。
- **三栏并排**：动态订单流（实时 · 5 秒刷新）/ 订单状态环形图（中心文字 "14 单"） / 预警雷达（4 类预警占比）。
- **PO 全程追溯**：进行中 + 逾期的 6 条订单，每张卡片带 7 节点进度点阵，点击进入订单详情。

**交互**
- 工序实时/今日/本周切换；预警卡片一键跳到预警中心；PO 卡片一键跳到订单详情。

---

### 4.2 ② 订单中心（Orders）
**定位**：订单全生命周期管理。

**功能要点**
- 4 个 KPI：本月订单 / 进行中 / 已延期 / 本月出货。
- 工具栏：新建订单 / 批量导入 / 导出 / 搜索 / 状态下拉 / 5 个状态 Tab（全部 12 / 进行中 / 未开始 / 已延期 / 已出货）。
- **订单列表**：12 列信息（订单号 / PO / 客户 / 数量 / 规格 / 交期 / 状态 / 紧急标签 / 进度条 / 操作）。
- 智能高亮：剩余 ≤3 天（琥珀），剩余 <0（玫红）。
- 进度条颜色：<50% 琥珀，<90% 蓝，≥90% 翠绿。
- 分页：每页 10/20/50 可选。

**订单详情弹窗（核心）**
- 左侧：订单号 + PO + 客户 + 款号 + 数量 + 尺码 + 出货日期。
- 中间：总进度卡（按 4 工序拆分百分比）+ **BOM 物料清单**（含损耗系数、订单用量、实际用量、已领用、库存对比）+ **7 步生产时间线**。
- 底部快捷操作：领料申请 / 采购计划 / 预览 / PO 标签 / 关闭。

**新建订单弹窗**
- 基础信息：订单号（自动）/ PO 号 / 客户 / 款式名称 / 开始日期 / 出货日期 / 工艺附件上传。
- **颜色 × 尺码矩阵**（4 色 × 5 码，含行/列合计，可手填）。
- BOM 物料清单（名称 / 单位 / 订单用量 / 损耗系数 / 实际用量）。
- 提交后：Toast「订单已创建，正在生成 BOM」。

**支持操作**：批量导入（CSV / Excel，模板下载）、导出（XLSX / CSV / PDF，字段可勾选）。

### 4.3 ③ 生产管理（Production）
**定位**：4 工序总览 + 排产可视化。

**功能要点**
- 4 张工序总览卡：裁剪 / 缝制 / 后整 / 包装（订单数、今日产量、效率、关键指标、渐变色背景）。
- 工序进度甘特：横向时间轴，色块表示工序占用，颜色按紧急度区分。
- 实时数据 + 3 个时间粒度切换（实时 / 今日 / 本周）。
- 支持 AI 排产建议入口（一键跳到 AI 智能决策）。

---

### 4.4 ④ 进销存（Inventory）
**定位**：采购计划 + 入库 + 领料 + 库存四位一体。

**4 个 KPI**：采购单总数 / 待采购物料（BOM 自动拆解）/ 采购金额 / 库存物料。

**4 个 Tab**
1. **采购计划**：采购单列表（采购单号 / 物料 / 供应商 / 数量 / 金额 / 预计到货 / 状态），状态：待采购 / 已下单 / 运输中 / 已到货；操作含「跟踪 / 催货 / 入库」。
2. **入库登记**：扫码 PO 号 + 物料选择 + 实际数量 + 库位 + 批次号 + 验收备注。
3. **领料出库**：批量发起，关联订单自动触发审批。
4. **库存概览**：物料列表（编号 / 名称规格 / 库位 / 当前库存 / 安全库存 / 关联 PO / 状态 / 补货 / 领料），3 状态：充足 / 偏低 / 紧急；带库存使用率进度条。

**库存详情弹窗**
- 物料基本信息 + 库存使用率大圆环（<40% 玫红警示，建议补货）。
- 出入库记录时间线（最近 5 条）。
- 一键「创建采购」自动提交审批。

**新建采购单弹窗**
- 采购单号（自动）+ 关联订单 + 供应商 + 到货日期 + 多物料清单（可加可删，单价 × 数量 = 行金额）。

### 4.5 ⑤ 版单开发（Pattern / 版房）
**定位**：AI 排料 + 3D 试身 + 批版闭环。

**功能要点**
- 流程条：客户需求 → AI 排料 → 样板制作 → 批版意见 → 核价 → 老板审核 → 客户确认（7 步可视化进度）。
- 工具栏：新建版单 / 上传算料（绿色高亮） / 概念串联（紫色）。
- 版单卡片列表：版单号 + 款号 + 名称 + 客户 + 制版师 + 状态（制版中 / 批版中 / 已审核 / 已确认）+ AI 排料标识。
- 状态徽章 + 状态颜色：amber / blue / emerald / purple。

**版单详情弹窗（核心）**
- 头部条：版单号 + 款号 + 名称 + 客户 + 制版师 + 状态。
- 7 步进度条（已完成 N/7 步）。
- **AI 智能排料卡**：用料率 / 损耗率 / 综合评分 / 生成耗时，AI 已生成 vs 未生成两种状态；按钮「启动 AI 排料 / 重新生成」（约 4 秒生成）。
- 工艺要求 / 物料清单 / 批版意见 / 客户确认附件上传区。
- 状态推进：制版师提交 → 业务员批版 → 老板审核 → 客户确认（流程触发审批通知）。

### 4.6 ⑥ 品控中心（QC）
**定位**：面料 / 辅料 / 中期 / 尾期四类检测 + 产前会议。

**4 个 KPI**：面料检测 / 辅料检测 / 中期查货 / 尾期查货。

**产前会议（PP Meeting）**
- 醒目横幅：下单前确认工艺、面料、辅料、风险点；「新建会议」入口。
- 主持：厂长 + 业务 + 版房 + 品控；会议纪要模板 + Checklist + 风险登记。

**报告卡片列表**（6 张示例）
- 类型（面料 / 辅料 / 中期 / 尾期）+ 物料名 + 检测日期 + 结果（合格 / 需关注）+ 备注。
- 「查看报告」入口（详情弹窗 + 改善建议 + 责任人 + 整改期限）。

### 4.7 ⑦ 财务结算（Finance）
**定位**：收入 / 成本 / 利润三维核算。

**4 个 KPI**：本月营收（¥286.4万 +15.6%）/ 本月成本 / 毛利（毛利率 30.8%）/ 待结算（¥42.6万）。

**6 类账单类型**（色块入口）
- 面辅料账单 / 裁剪账单 / 缝制车间 / 后整理账单 / 特种工艺 / 工资结算。
- 点击进入对应账单列表（结算单号、客户、应收、已收、未收、开票日期、状态）。

**订单利润排行 TOP 5**：横向条形图 + 毛利率 + 利润金额。
**成本结构占比**：Chart.js 环形图（面料 / 人工 / 辅料 / 外协 / 其他）。

### 4.8 ⑧ 审批中心（Approval）
**定位**：单据审批 + 流程自定义。

**4 个 KPI**：待我审批 / 我发起的 / 已通过 / 审批总数。

**4 个 Tab**：待审批 (5) / 我发起的 (8) / 已通过 / 已驳回。

**审批类型**：领料（橙）/ 采购（蓝）/ 外协（紫）/ 请假（翠），紧急标识 + 状态徽章（待审批 / 已通过 / 已驳回）。

**审批详情弹窗**
- 申请编号 + 申请类型 + 申请时间。
- 申请内容区 + 审批流程（节点 + 处理人 + 时间 + 意见）。
- 底部操作：同意 / 驳回 / 转发 / 加签。

**新建审批**：选择类型 → 关联单据 → 填写说明 → 选审批人 → 提交。

### 4.9 ⑨ 外协管理（Outsourcing）

**4 个 KPI**：外协订单 / 外协供应商 / 外协金额 / 外协账号（合规监控 6/6，超限 ¥988/个/年）。

**外协订单表**：外协单号 / 类型 / 供应商 / 数量 / 金额 / 关联 PO / 回厂日期 / 状态（待开始 / 加工中 / 已完成）。

**新建外协单**：选择工序类型 + 外协供应商 + 关联 PO + 数量 + 金额 + 回厂日期 + 结算方式。

### 4.10 ⑩ 分销中心（Distribution · NEW）
**定位**：扫码注册 + 线上支付 + T+N 自动结算。

**关键能力**
- **佣金模式配置**（3 选 1）
  - 按订单金额百分比：5%
  - 按件固定佣金：¥2/件
  - 阶梯式佣金：5%-8%
- **分销账号表**：分销商 + 等级（金 / 银 / 铜）+ 订单数 + 订单金额 + 佣金 + 状态（正常 / 待审核）。
- **隐藏设置**（仅超管可见）：佣金模式 + 提现审批 + 黑名单。

### 4.11 ⑪ AI 智能决策（AI · 决策中枢）
**定位**：基于 12 万条历史订单 + 实时生产数据，为管理层提供决策支持。

**顶部状态条**：AI 引擎在线 + 8 个模型加载 + 推理延迟 1.2s + 3 个核心指标（86% 准确率 / +18% 效率 / 2.4k 今日决策）。

**两大主区块**
1. **产能预测**：进行中订单列表，每条显示当前进度 + 预计完成日期 + 进度条。
2. **AI 排产建议**：5 条 P0~P4 优先级排序（含「物料齐全 + 交期紧」「待物料 + 远期」等理由标签）。

**产能基础数据滑块**（AI 排期参数）
- 4 工序：裁剪 / 缝制 / 后整 / 包装，每项带 件/天 阈值与可调滑块。
- 一键「重算工期」→ Toast 确认。

**本周关键建议**（4 类）
- 工序调整建议（包装 D 组效率 76%，建议从缝制 C 调 2 名熟练工支援）。
- 订单预测（6 月第三周订单量 +20%，建议提前备料）。
- 标杆组推广（缝制 B 组 5 天第一 → 推广工艺）。
- 价格波动预警（聚酯面料 +3.2%，建议锁定 Q3 合同节省 ¥8.6 万）。

**导出能力**：AI 排料方案可导出 PDF / Excel。
**近 7 日产能趋势**：Chart.js 面积图。

### 4.12 ⑫ 预警中心（Alerts）
**定位**：4 类预警统一入口 + 规则配置。

**4 个 KPI**：出货预警 / 库存预警 / 开裁预警 / 采购到货。

**预警消息列表**
- 类型徽章 + 等级（高 / 中 / 低，对应玫红 / 琥珀 / 紫）+ 触发方式（自动 🤖 / 手动 👆）+ 紧急标签。
- 标题 + 描述 + 时间 + 关联 PO + 「立即处理」/「忽略」按钮。

**预警类型规则**（可配置）
- 库存预警阈值：安全库存的 80%（默认开启）。
- 出货预警提前：剩余 3 天。
- 开裁预警提前：计划开裁日前 2 天。

**自动触发动作示例**
- 库存偏低 → 一键创建采购单。
- 出货临近 → 跳到订单详情。
- 物料未齐 → 跳到开裁预警详情。

**自定义字段**（5 个示例）
- 字段名 + 类型（文本 / 枚举 / 图片 / 数字）+ 使用范围 + 可见角色 + 必填。

## 5. 横向系统能力
